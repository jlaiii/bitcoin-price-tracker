About

Bitcoin Price Tracker is a lightweight, client-only dashboard showing real-time BTC price data from CoinGecko. No server, no tracking, no API keys required.

Features include interactive price charts, a DCA simulator, mining statistics, fear & greed sentiment gauge, market cap comparisons, and historical milestone data.

Built with vanilla HTML5, CSS3, and JavaScript. Charts powered by Chart.js.

Data from CoinGecko. Updates every 60 seconds.

Pure client-side — no server, no tracking, no API keys needed.
